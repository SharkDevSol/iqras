# IQRAS Super Admin Dashboard

Multi-Branch School Management System - Centralized dashboard for managing multiple school branches.

## Features

- 🏫 Dynamic branch management (add/remove branches without coding)
- 📊 Real-time dashboard with all branches data
- 👥 Students & Staff management across all branches
- 💰 Finance tracking and comparison
- 📚 Academic data (marks, evaluations, subjects)
- 📈 Branch performance comparison
- 🔍 Automatic health monitoring
- 🔐 Secure JWT authentication

## Quick Start

### Backend

```bash
cd backend
npm install
npm start  # Auto-creates database and tables
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
